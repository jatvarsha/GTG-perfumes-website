# GTG Perfumes Website

A pixel-perfect, responsive landing page for GTG Perfumes with clean separation of HTML, CSS, and JavaScript.

## Features

- **Responsive Design** - Works seamlessly on desktop, tablet, and mobile devices
- **Interactive Product Gallery** - Image carousel with thumbnails, arrows, and dot navigation
- **Subscription Modal** - Popup for Single and Double subscription options with fragrance selection
- **Animated Stats Counter** - Numbers count up when scrolled into view
- **Comparison Table** - Feature comparison with competitor brands
- **Expandable Accordion** - Collection items with smooth expand/collapse
- **Newsletter Signup** - Footer newsletter subscription form

## Technologies

- HTML5
- CSS3 (Flexbox, Grid, Animations)
- Vanilla JavaScript (No frameworks)

## File Structure
```
├── index.html          # Main HTML structure
├── style.css           # All styling and responsive design
└── script.js           # Interactive functionality
```

## Functionality

- Dynamic Add to Cart with 9+ product variations
- Modal popup for subscription selection
- Smooth scroll animations
- Hamburger menu for mobile
- Interactive fragrance selector

## Installation

1. Clone the repository
```bash
git clone https://github.com/yourusername/gtg-perfumes.git
```
2. Open `index.html` in your browser
3. Replace placeholder images with actual product images from Figma
